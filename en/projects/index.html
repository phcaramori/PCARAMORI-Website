Minecraft Discord Bot

Minecraft Discord Bot

A bot I made to facilitate minecraft server owners in integrating their server to their discord server. From discord, users can check who is currently on-line, check server commands, and other useful information such as IP and seed. Made with Discord.JS, still WIP. Currently, only works to display the status of a single server - I'm working on adding a db so this can be set on a server-by-server basis.
