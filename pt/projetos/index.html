Bot do Discord

Bot do Discord

Um bot que fiz para facilitar a integração de servidores do Minecraft ao Discord. No Discord, os usuários podem verificar quem está online no momento, verificar comandos do servidor e outras informações úteis, como IP e seed. Feito com Discord.JS, ainda work-in-progress. Atualmente, funciona apenas para exibir o status de um único servidor - estou trabalhando para adicionar um banco de dados para que os comandos possam ser definidos individualmente para cada servidor

Flappy Bord

Uma réplica do Flappy Bird que fiz, utilizando, novamente, apenas HTML, JS e CSS simples. Com 4 níveis de dificuldade e um contador de high-score, o jogo usa o método getBoundingClientRect para calcular as colisões entre objetos.
